# pagarme

## Requirements

 - PHP >= 7.0
 - Composer
 - sqlite3
 
## How to use it

 - Clone repository
    `git clone https://github.com/jameswpm/pagarme`;
 
 - Install dependencies
    `composer install`
    
 - Run built in server in src directory
    `cd src/ && php -S localhost:8089/app`
    
 - Access the localhost:8089 in your browser and test the page
